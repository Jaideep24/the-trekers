Refund and Cancellation

Please Read Carefully the Cancellation and Refund Rules – for Domestic and National Tours / Treks / Camping / Events by Treks and Trails India

 

Cancellation and Refund Policy:

 

Events (1 or 2 days)

  1. 75% refund if notified via phone conversation 8 or more days prior to the event date.
  2. 50 % refund if notified via phone conversation 4 to 7 days prior to the event date.
  3. No refund if the cancellation requested is less than 3 days prior to event date
  4. No show No Refund.
  5. Event Tickets cannot be transferred to another date against cancellation.
  6. Event Tickets cannot be transferred to another person against cancellation.
  7. If Trek get cancelled, we will refund "Trek Amount" only.

 

Events (3-4 Days)

  1. If cancellation done prior to 30 days of outing 75% will be refunded.
  2. If done prior to 15 days of outing 50% will be refunded
  3. If done prior to 10 days of outing 30% will be refunded
  4. After that no refund will be provided
  5. No show No Refund
  6. Event Tickets cannot be transferred to another date against cancellation
  7. Event Tickets cannot be transferred to another person against cancellation
  8. If Trek get cancelled, we will refund "Trek Amount" only

 

Events (More than 4 Days)

  1. If cancellation done prior to 45 days of outing 75% will be refunded.
  2. If done prior to 30 days of outing 50% will be refunded
  3. If done prior to 15 days of outing 30% will be refunded
  4. After that no refund will be provided
  5. No show No Refund
  6. Event Tickets cannot be transferred to another date against cancellation
  7. Event Tickets cannot be transferred to another person against cancellation
  8. If Trek get cancelled, we will refund "Trek Amount" only

 

Dudhsagar Waterfall Trek

  1. If cancellation done prior to 30 days of outing 75% will be refunded
  2. If done prior to 15 days of outing 50% will be refunded
  3. After that no refund will be provided
  4. No show No Refund
  5. Event Tickets cannot be transferred to another date against cancellation
  6. Event Tickets cannot be transferred to another person against cancellation
  7. If Trek get cancelled, we will refund "Trek Amount" only
